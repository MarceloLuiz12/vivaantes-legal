Política de Privacidade

Última atualização: maio de 2025

O VivaAntes foi criado com um princípio simples: seus dados são seus. Esta política explica de forma direta o que coletamos (spoiler: quase nada) e por quê.

1. Dados armazenados no seu dispositivo

Todas as suas experiências, listas e preferências ficam salvas exclusivamente no seu dispositivo, usando um banco de dados local. Nenhum dado pessoal é enviado para servidores externos.

2. Dados de diagnóstico (Firebase)

Utilizamos o Firebase Crashlytics para receber relatórios de falhas anônimos quando o app trava. Esses relatórios não contêm nenhuma informação que identifique você pessoalmente — apenas o stack trace do erro e informações do dispositivo (modelo, versão do SO).

Também utilizamos o Firebase Analytics para entender quais telas são mais usadas. Os eventos coletados são agregados e anônimos.

3. O que não coletamos

  • Nome, e-mail ou qualquer dado de conta
  • Localização
  • Contatos ou fotos
  • Conteúdo das suas experiências ou listas

4. Compartilhamento com terceiros

Não vendemos, alugamos nem compartilhamos seus dados com terceiros para fins comerciais. Os dados de diagnóstico são processados pela Google LLC (Firebase) de acordo com a Política de Privacidade do Google.

5. Backup

O app oferece a funcionalidade de backup manual, que exporta seus dados para um arquivo no seu próprio dispositivo. O destino do arquivo é sempre escolhido por você.

6. Crianças

O VivaAntes não é direcionado a crianças menores de 13 anos e não coleta dados de crianças de forma intencional.
